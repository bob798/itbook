# Hazelcast教程
Hazelcast是一个用Java编写的基于内存的数据网格管理平台（open source in-memory data grid）

## 功能
### 支持的数据结构

#### Distributed Caching
1. AtomicLong
2. AtomicReference
3. CountDownLatch
4. IdGenerator
5. List
6. Lock
7. Map
8. MultiMap
9. Queues
10. Replicated Map
11. Ringbuffer
12. Semaphores
13. Set
14. Topic and ReliableTopic
